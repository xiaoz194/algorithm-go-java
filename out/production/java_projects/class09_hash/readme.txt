hash数据结构专题

hash的特性
1） 输入 无穷尽； 输出 有穷尽
2）same in =》 same out  相同的输入一定得到相同的输出
3）diff in may be =》 same out 不同的输入可能会得到相同的输出（很好理解，输入无穷，输出有穷，那肯定会出现不同输入是相同输出的情况，但是概率较小，称为hash碰撞）
4）均匀性，离散性
5）hash得到的结果%m，在0～m-1上也一定具有均匀性和离散性（缩小范围，性质依然相同

布隆过滤器 - 爬虫去重问题 黑名单问题   适合所有大数据量的 添加，查询 且没有删除的问题

可以极大的减少内存，但是失误率不可避免，但是可以做到极低的失误率

对于黑名单过滤，有两种失误类型：
1） 本身是黑名单，结果认为是白名单过了  这种情况不会发生
2）本身是白名单，结果认为是黑名单被拦截了 这种情况会发生，但是概率较低，且可以接受

布隆过滤器的实现方式： 位图 （bit array 或叫 bit map）
